# vue-my-lib

## Docker compose
```
docker-compose up
```
or if you want to rebuild
```
docker-compose up --build
```
Open `http://localhost:8080` or `http://192.168.99.100:8080` for Windows

## Project setup (dev)
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
